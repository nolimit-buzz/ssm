## Usage Instructions

### For Gmail/Outlook/Apple Mail:

1. Open the HTML file in a web browser
2. Select all content (Cmd/Ctrl + A)
3. Copy (Cmd/Ctrl + C)
4. In your email client settings, go to Signature settings
5. Paste the copied HTML into the signature field
6. Save

### For Outlook (Desktop):

1. Open Outlook
2. Go to File > Options > Mail > Signatures
3. Create a new signature
4. Click "New" and paste the HTML content
5. Save

### Customization

To customize for a new team member:

1. Open `generic-signature-template.html`
2. Replace `[Full Name]` with the person's name
3. Replace `[Job Title]` with their role
4. Replace `[email]@swapstation.mobility` with their actual email address
5. Update phone number if different
6. Save as `[firstname-lastname]-signature.html`
